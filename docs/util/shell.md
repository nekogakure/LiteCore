この章では、シェル機能について記述します。
実装されているファイルは[shell.c](../../src/kernel/util/shell.c), [shell.h](../../src/include/shell/shell.h)です。

## 概要
シェルモジュールは、対話型コマンドラインインターフェースを提供します。ユーザー入力の読み取り、バッファ管理、コマンドの実行、プロンプト表示を行います。バックスペース、Enterキー、制御文字の処理をサポートします。

## 関数 / API

#### `void init_shell(void)`
シェルを初期化します。ウェルカムメッセージを表示し、コマンドシステムを初期化し、プロンプトを表示します。

引数: なし

#### `void shell_run(void)`
シェルのメインループを実行します。キーボード入力を読み取り、コマンドを実行し続けます（ブロッキング）。

引数: なし

#### `int shell_readline_and_execute(void)`
1行のコマンドを読み取って実行します（ノンブロッキング）。1文字ずつ読み取り、Enterが押されたらコマンドを実行します。

引数: なし

戻り値: 処理した文字数

## 定数 / 定義

- `SHELL_BUFFER_SIZE`: 256 - シェル入力バッファのサイズ
- `SHELL_PROMPT`: "LiteCore> " - シェルプロンプト文字列

## 構造体

このファイルには公開構造体の定義はありません。

### 内部実装

- **入力バッファ**: 最大256文字のコマンドラインを保持
- **バッファ位置**: 現在の入力位置を追跡
- **プロンプト表示**: コマンド実行後に自動的にプロンプトを表示

### キー処理

- **Enter（\n, \r）**: コマンドを実行し、バッファをクリア
- **バックスペース（\b, 0x7F）**: 最後の文字を削除
- **制御文字（^C, ^D）**: 特別な処理（キャンセル、EOF）
- **通常文字**: バッファに追加し、画面に表示

### コマンド実行

シェルは`execute_command()`関数を使用してコマンドを実行します。コマンドシステムが登録されたコマンドハンドラを検索し、適切な関数を呼び出します。
