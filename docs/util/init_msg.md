この章では、カーネル初期化処理について記述します。
実装されているファイルは[init_msg.c](https://github.com/nekogakure/LiteCore/blob/main/src/kernel/util/init_msg.c), [init_msg.h](https://github.com/nekogakure/LiteCore/blob/main/src/include/util/init_msg.h)です。

## 概要
カーネル初期化モジュールは、システムの起動時に必要なすべてのサブシステムを初期化します。メモリ管理、割り込み処理、デバイスドライバ、ファイルシステムを順番に初期化し、システムを使用可能な状態にします。

## 関数 / API

#### `void kernel_init()`
カーネルのサブシステムを初期化します。以下の順序で初期化を実行します：

1. メモリ管理の初期化（`memory_init()`）
2. 割り込みシステムの初期化（`idt_init()`, `interrupt_init()`）
3. タイマー割り込みハンドラの登録（IRQ 0）
4. デバイスドライバの初期化（キーボード）
5. 割り込みの有効化

引数: なし

### 初期化シーケンス

1. **メモリ管理**: 物理メモリマップとヒープを初期化
2. **割り込みシステム**: IDTとPICを設定し、割り込みハンドラを登録
3. **タイマー**: システムタイマー（IRQ 0）のハンドラを登録
4. **キーボード**: PS/2キーボードドライバを初期化
5. **割り込み有効化**: STI命令で割り込みを有効化

各ステップで"ok"メッセージを出力し、初期化の進行状況を表示します。
