この章では、エラーメッセージ表示機能について記述します。
実装されているファイルは[msg.c](../../../src/kernel/util/msg.c)です。

## 概要
エラーコードから対応するエラーメッセージを表示するユーティリティ機能を提供します。統一されたエラーメッセージフォーマットにより、デバッグを容易にします。

## 関数 / API

#### `void msg_from_code(int error_code)`
エラーコードに対応するエラーメッセージをコンソールに表示します。

引数:
  - error_code(int): エラーコード

## 定数 / 定義

- `MSG_NO_ERROR`: 0 - エラーなし
- `MSG_INTER_ERROR`: 1 - 割り込み処理エラー
- `MSG_MEM_ERROR`: 2 - メモリ管理エラー
- `MSG_FS_ERROR`: 3 - ファイルシステムエラー
- `MSG_IO_ERROR`: 4 - I/O操作エラー

## 構造体

このファイルには構造体の定義はありません。

## 実装の詳細

### エラーコードマッピング
各エラーコードは特定のサブシステムのエラーに対応します：

- **0 (MSG_NO_ERROR)**: エラーなし。正常終了を示します
- **1 (MSG_INTER_ERROR)**: 割り込みハンドラや割り込み管理でのエラー
- **2 (MSG_MEM_ERROR)**: メモリアロケーションやページング関連のエラー
- **3 (MSG_FS_ERROR)**: ファイルシステム操作のエラー
- **4 (MSG_IO_ERROR)**: デバイスI/O操作のエラー
- **その他**: 未知のエラーコードとして表示

### 使用例
```c
int result = some_operation();
if (result != 0) {
    msg_from_code(result);
}
```

### 拡張性
新しいエラータイプを追加する場合、新しい定数を定義し、`msg_from_code()`のswitch文にcaseを追加するだけで済みます。
