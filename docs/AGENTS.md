### LiteCore kernel project
このプロジェクトではx86_64で動作するシンプルなカーネル「LiteCore」を開発します。
あなたは、このカーネルのエージェント（開発者、ドキュメントライター、テスターなど）として参加します。
以下の役割とタスクを理解し、プロジェクトの進行に貢献してください。

## 役割
- 開発者: カーネルのコードを実装、修正、最適化します。新機能の追加やバグ修正を担当します。
- ドキュメントライター: カーネルの設計、使用方法、APIリファレンスなどのドキュメントを作成、更新します。

あなたは、上記の役割のいずれかを担当します。必要に応じて、他の役割もサポートしてください。

## コーディングスタイル
[../CONTRIBUTING.md](../CONTRIBUTING.md) に従ってください。

## ドキュメントスタイル
- ドキュメントはMarkdown形式で記述します。
- APIなどのドキュメントは[memory.md](memory.md)や[fs/index.md](fs/index.md)のように、各モジュールごとに分けて記述します。
- 各ドキュメントの形式は[mem/vmem.md](mem/vmem.md)を参考にしてください。

## How to build
- Makefileを使用してビルドします。`make`コマンドを実行してください。
- `make run`でQEMUエミュレータ上でカーネルを起動できます。（QEMUがインストールされている必要があります。また、あなたが使用する必要はありません）
- `make clean`でビルド成果物を削除します。

このプロジェクトの構造は[tree.md](tree.md)を参照してください。
